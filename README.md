# 🚀 NavesUSFX_012026 - Laboratorio C++ Unreal Engine 5

## 📌 Descripción
Proyecto individual que implementa un sistema de enemigos en C++ para Unreal Engine 5.
- Clase `AEnemy`: Actor con movimiento autónomo por waypoints.
- Clase `ASpawnerEnemigos`: Genera instancias en `BeginPlay` vía `SpawnActor`.

## 🛠️ Requisitos
- Unreal Engine 5.7
- Visual Studio 2022 Community
- Windows 10/11 (64-bit)

## ▶️ Cómo Ejecutar
1. Clonar repositorio.
2. Abrir `NavesUSFX_012026.uproject` con Unreal Engine 5.
3. Colocar `SpawnerEnemigos` en el nivel y asignar `ClaseEnemigo = Enemy_C`.
4. Presionar `Play`.
